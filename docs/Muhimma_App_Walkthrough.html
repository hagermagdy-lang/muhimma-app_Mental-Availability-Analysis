Overview

What Is Muhimmaapp Analytics?

Muhimmaapp Analytics is a browser-based tool that transforms raw consumer survey data into instant brand health insights using the Mental Availability framework from the Ehrenberg-Bass Institute.

Upload a CSV → the app automatically calculates Mental Penetration, Network Size, and Mental Market Share for every brand in your category — displayed across 5 interactive screens, including an AI-powered analyst. No Excel, no manual calculations, no backend server. Everything runs inside the browser.

Section 01

The 5 Screens

1

📥 Import — Data Upload

Route: /
The starting point. Users drag-and-drop or browse for a .CSV survey file.
  • Accepts Wide Text CSV format (columns = CEPs, cells = brand names)
  • PapaParse reads the file entirely in the browser — no data is sent to any server
  • After upload, a preview screen shows the detected brands and CEPs
  • User confirms to proceed to the dashboard
⚠️ Known issue: Data is lost on page refresh. A fix using sessionStorage is planned in Phase 1 of the roadmap.
2

📊 Summary — Executive Dashboard

Route: /executive-summary
High-level view of all brands ranked by Mental Market Share (MMS).
  • Avg Category MPen — average penetration across all brands
  • Average Network Size — avg CEPs linked per respondent
  • Total Respondents — count of survey rows processed
  • Brand Performance Table — all brands sorted by MMS with inline progress bars for MPen
3

🏷️ Brands — Brand Deep Dive

Route: /brand-analysis
Deep-dive into any single brand selected from a dropdown.
  • Hero number: brand's Mental Penetration %
  • Network Size score with a visual strength bar
  • Market Sentiment label: Strong Positive / Stable / Vulnerable
  • Comparison vs. category average for MPen and NS
  • Top 4 CEPs for the brand shown as percentage bars
⚠️ The "Internal Penetration Trend" area is currently a placeholder. A Recharts radar chart is planned for Phase 2.
4

🗺️ CEP Grid — Heatmap Matrix

Route: /cep-grid
A full heatmap where rows = brands and columns = CEPs. Each cell shows the association strength.
  • Darker cell = stronger brand–CEP association
  • Color alternates between primary (orange) and secondary (blue) per brand row
  • Bottom widgets: total CEPs, total data points, dominant brand
  • Export CSV button (UI only — functionality coming in Phase 3)
⚠️ Formula issue: Percentages currently show distribution within a brand, not true penetration across all respondents. This is a planned Phase 1 fix.
5

🤖 AI Analyst — Automated Brand Insights

Route: /ai-insights
Generates strategic brand intelligence using a local Ollama (Llama 3) instance. Analyzes your uploaded data and produces a written summary.
  • Statistical Anomalies — highlights unusual metric patterns (e.g., high penetration but low network size)
  • Marketing Strategy — recommends specific growth actions for top brands
  • Runs entirely locally via Ollama — no data leaves your machine
  • Results appear as a single formatted block after inference completes
⚠️ Requires Ollama running locally. Set OLLAMA_ORIGINS="*" in PowerShell before starting Ollama to allow browser access.
Section 02

Key Metrics Explained

MetricFormulaWhat It Means
Mental Penetration (MPen) Respondents who mentioned the brand ÷ Total respondents × 100 The % of the market that has the brand in mind. The most important top-line metric.
Network Size (NS) Total brand associations ÷ Respondents who mentioned the brand How many buying situations (CEPs) the brand is mentally linked to, on average, per person who knows it.
Mental Market Share (MMS) Brand total associations ÷ All category associations × 100 The brand's share of all mental connections in the category. Comparable to traditional market share but measured in minds.
Section 03

CSV Format Guide

The app expects a Wide Text format. Each column is a CEP, each row is a respondent, each cell is the brand(s) they mentioned for that CEP.

Respondent ID, When I'm celebrating..., When I'm tired..., When I'm at work...
R001,          Pepsi,                   Nescafe,           Red Bull
R002,          Coca-Cola,               Pepsi,             Nescafe
R003,          Pepsi,                   Red Bull,
R004,          ,                        Nescafe,           Pepsi
  • Row 1 = CEP column headers (long phrases are auto-trimmed to 5 words)
  • Column 0 = Respondent ID (ignored in calculations)
  • Multiple brands in one cell: separate with a comma Pepsi, Coca-Cola
  • Empty cell = no association for that CEP/respondent (perfectly valid)
